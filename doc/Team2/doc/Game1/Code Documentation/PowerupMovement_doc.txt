PowerupMovement - Lew

Variables: sets speed for the powerups, keep track if it is visible or not

Update(): Moves the powerup across the screen by the speed declared

OnBecameVisible(): Sets the visible variable to true

OnBecameInvisible(): Once the objects goes offscreen, it will be destroyed and visible variable will
be set back to false.  This is to prevent unnecessary memory usage.

OnTriggerEnter(): If the powerup collides with the player, destroy the powerup.
